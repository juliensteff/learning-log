# CS50P — Harvard's Introduction to Programming with Python

Coursework for [CS50P](https://cs50.harvard.edu/python/). One folder per lecture / problem set.

## Convention
- `lectureN/` — code written while following along with Lecture N.
- `psetN/` — solutions to Problem Set N (when I get there).
- Each file is a small, runnable script. Comments at the top note what the lecture covered.

## Why this is separate from `experiments/` and `projects/`
- `cs50p/` is coursework — guided exercises, not original work.
- `experiments/` is my own throwaway exploration.
- `projects/` is shipped portfolio pieces with real READMEs and demos.

Recruiters scan repo structure. Keeping these separate makes the signal clean.
